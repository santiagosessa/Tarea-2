# Registro al Evento - TP JavaScript Esencial

## Descripción breve del proyecto
Este proyecto es una aplicación web sencilla que simula un formulario de registro para un evento. Implementa validación básica del lado del cliente utilizando JavaScript puro (Vanilla JS) para determinar si un usuario cumple con el requisito de ser mayor de edad (>= 18 años) para acceder al evento, mostrando un mensaje dinámico con colores correspondientes.

## Instrucciones para ejecutar
1. Clona este repositorio en tu computadora usando Git:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. No es necesario instalar dependencias.
3. Abre el archivo `index.html` en tu navegador web preferido (Google Chrome, Firefox, Safari, Edge, etc.) haciendo doble clic sobre él.
